THE MCU API is a modern, scalable RESTful API built using **Python**, **Django**, and **Django REST Framework (DRF)**.  
It integrates **Tailwind CSS** for clean UI styling and **Swagger (OpenAPI)** for interactive API documentation.

This project follows best practices for maintainability, scalability, and developer experience.
